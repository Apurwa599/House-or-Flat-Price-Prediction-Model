## House-or-Flat-Price-Prediction-Model
### Overview
This project focuses on building a Machine Learning model to predict house or flat prices based on various features such as location, size, number of rooms, and other property-related attributes. The project was developed as part of my Data Science / Machine Learning learning and internship work.Accurate price prediction helps buyers, sellers, and real estate companies make informed decisions.

### Objectives
* Analyze real estate data
* Perform data cleaning and preprocessing
* Build a machine learning model for price prediction
* Evaluate model performance
* Predict house/flat prices for new inputs
